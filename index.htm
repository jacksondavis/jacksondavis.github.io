Capital One Summit for Software Engineers

This is my my application for the Capital One Summit for Software Engineers challenge described here.

When I first heard of the Capital One Summit for Software Engineers, I became excited for both the program itself and the challenge we had to complete in order to gain acceptance.

The challenge required us to pull posts tagged with #CapitalOne from Instagram, display this data, and create some sort of optional visualization of the sentiment associated with each post. This app was created using Flask, AlchemyAPI, InstagramAPI, and d3.js. A detailed description of how the application works is on my Github.

Watdo

This application was created alongside fellow UIUC students Aadil Bhatti and Daniel Margosian at NIU's HuskieHackk 2015.

Watdo is a web-application built during NIU's HuskieHack 2015 in order to address the city of Dekalb's request for a way to aggregate and organize community events. Watdo was built on the Meteor.js framework, and makes use of several RSS feeds as well as the Eventify API. I was primarily responsible for creating the application's frontend, and also contributed to areas in the backend and helped determine overall design decisions. This application was built alongside fellow UIUC computer science students Aadil Bhatti and Daniel Margosian.
